# Joy to the World

A static archive of the birthday wall made for Joy's 50th birthday.

This site preserves all of the approved messages and photos as a permanent front-end archive. The original Supabase-backed guestbook has been removed, and the content now lives directly in this repo so the site can stay online as a read-only memory wall.

Live site: [joy-to-the-world.vercel.app](https://joy-to-the-world.vercel.app/)
